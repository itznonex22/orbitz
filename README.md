# Orbitz

A responsive website of a solar system to visualize and simulate the [Orbitz](https://orbitz.ntabucejo.vercel.app/) in 2D!

## Strategies 

* CSS Flexbox
* CSS Grid
* Geometry sizing with (%)
* Animation using Framer Motion
* CSS (Fixed / Absolute) Positions
